# 🚀 Saksham's 3D Developer Portfolio

Welcome to my personal 3D interactive developer portfolio! This project showcases my background in Computer Science Engineering, my skills in backend systems and AI development, and my key projects using a modern Light Theme Glassmorphism aesthetic.

## 🌟 Features
- **3D Interactive Elements**: Built using Three.js and React Three Fiber to display custom 3D models and interactive scenes.
- **Light Theme Glassmorphism**: A sleek, modern UI design featuring vibrant gradients, translucent frosted glass effects, and dark contrast typography.
- **Dynamic Content**: Sections for My Education (VIT Bhopal), detailed showcases of my top projects, and an interactive "About Me" 3D globe.
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop views using Tailwind CSS.
- **Functional Contact Form**: Integrated with EmailJS for direct communication.

## 🛠️ Tech Stack
- **Framework**: React.js (Vite)
- **3D Rendering**: Three.js, `@react-three/fiber`, `@react-three/drei`
- **Styling**: Tailwind CSS
- **Animations**: GSAP (GreenSock)

## 📂 Featured Projects
- **Security Configuration Compliance Auditor (Java)**
- **Enterprise GRC Implementation Simulation**
- **AI-Powered Document Analyzer**
- **Brain Tumor Detection System**

## 💻 Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Saksham-n/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

Open `http://localhost:5173` to view it in the browser.
